Puget Sound Lost & Found

This was my senior capstone project I worked on with four other computer science classmates. We worked
closely with the University of Puget Sound's security services to modernize their lost and found system.
Previously, security services kept track of lost items around campus using an excel spreadsheet. For
community members the only way to tell if security had found any lost items was by physically going to security.
By making this app, students and faculty can easily check to see if their lost item had been recovered. The features
of this app include account creation & login with email validation, creating/displaying/deleting posts, visible user profiles.
The app is written in Javascript, HTML5, and CSS with use of AWS Lightsail, AWS S3, MongoDB, Axios, and Bootstrap.

Glizzy Royale

Glizzy Royale is a browser based multiplayer game inspired by internet memes and the popular phone game,
Brawl Stars. Players connect through the browser and get put into an arena with all other players currently connected
to the server. In the arena players battle in a free for all style as hot dogs (glizzies) and can fire ketchup blobs to
deal damage to other players. When a player's health is running low the golden glizzy in the center of the arena dispenses
bread crumbs to replenish players' buns (health). Written in C# using Unity Game Engine and the WebGL plugin to convert to javascript
formerly hosted on an apache server.

Schelling's Model of Self Segregation

Schelling's model uses a grid with different colors to represent two populations. User's can then set more
options such as the threshold (% of same neighbors). When the user has set everything how they would like and run
the program, the individual chunks in the grid move around until they are in an arrangement where everyone is satisfied.
Schelling's model is used to show that groups will naturally self-segregate themselves to certain degrees. This is written
in javascript, HTML5, and CSS.

Java Projects

These java projects were assignments throughout my 4 years pursueing my Bachelor's degree at the University of Puget Sound.

Movie Analyzer

The movie analyzer program reads raw movie review data a file and cleanses the data to then be used to construct
graphs. The program runs in the command line and prompts users with two options for how to construct the graph,
after the graph is built (roughly 4 seconds) the user has options to see either statistics about the graph or
individual movie nodes. In completing this assignment me and my partner had to use a graph class we had written
previously and required us to implement Floyd-Warshall's algorithm and Djikstra's algorithm for some features.
Written completely in Java.

Directed Graph

This is a directed graph in java implementing java's graph interface. The graph is stored using a hashmap of lists
where each key is a node and each value is an arrayList of the key's neighboring nodes.

Priority Queue

The priority queue works similar to a queue but now items being stored also have a priority assigned to them.
The highest priority items go to the top of the queue. Each item has either no children (leaf nodes), or they
have either a left, or both a left and right child. Because of the way the priority queue is built, nodes cannot
have just a right child.

Evil Hangman

Graph Coloring In Haskell

This project tackles the classic graph coloring problem using the functional programming language, Haskell.
The graph coloring problem is where there is a graph and we want to assign colors to all the nodes.
Neighboring nodes cannot be assigned the same color and we want to try using as few colors as possible to
give a satisfactory color assignment to each node.
