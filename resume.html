Technical Projects

Evil Hangman

The evil hangman project was an assignment from CSCI 261 where I had to write a version of the classic game hangman but with a catch, the game was unwinnable. The game starts as a word and with each guess the computer would change the word being guessed making sure that it fit with any correctly guessed letters and was the appropriate length.

Black Jack Simulator

The black jack simulator project was an assignment from CSCI 161 where I had to write a game for users to play the classic casino game, black jack. This was my first big project I did as a programmer, I had to create multiple classes for cards, the deck, and the game along with creating an ai dealer so players could play against the computer as the house. This project added additional fuel to my growing passion for programming.
