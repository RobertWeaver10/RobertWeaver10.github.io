Graph Coloring In Haskell

This project tackles the classic graph coloring problem using the functional programming language, Haskell. The graph coloring problem is where there is a graph and we want to assign colors to all the nodes. Neighboring nodes cannot be assigned the same color and we want to try using as few colors as possible to give a satisfactory color assignment to each node.
